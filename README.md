# Public Crypto Wallet Widget

A simple, easy-to-use crypto wallet widget displaying 7 popular cryptocurrencies with live USD prices. Users can copy wallet addresses or check balances on blockchain explorers. Ideal for GitHub Pages or personal websites.

## Features
- Supports Bitcoin (BTC), Ethereum (ETH), Binance Coin (BNB), Solana (SOL), Litecoin (LTC), Dogecoin (DOGE), Tron (TRX)
- Real-time prices via CoinGecko API
- Copy wallet addresses with one click
- Open wallet in blockchain explorers
- Clean dark-themed UI with hover effects
- Auto-refreshes every 30 seconds

## Setup on GitHub Pages
1. **Clone the repository** to your GitHub account (fork or copy it).  

2. **Enable GitHub Pages**
   - Go to your repository **Settings â†’ Pages**
   - Under **Branch**, switch from `None` to `main` and click **Save**
   - Wait about 30 seconds to a minute, then refresh the page
   - Your public link will be available for use  

3. **Edit the widget**
   - Open `index.html` and update the `ADDRESSES` object with your wallet addresses  
   - Update the `EXPLORERS` object with links to the public blockchain explorers if needed  

4. **Your widget is live**
   - Any changes you make to `index.html` will reflect after committing and refreshing your GitHub Pages URL
